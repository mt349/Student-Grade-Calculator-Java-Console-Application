# Student Grade Calculator - Java Console Application

## Description
This is a simple **Java console application** that calculates the **total marks**, **percentage**, and **grade** of a student based on the marks entered for multiple subjects. The program uses **loops** to gather input for multiple subjects and **switch statements** to determine the grade.

---

## Features
- Input the number of subjects a student has.
- Enter the name and marks for each subject.
- Calculates the **total marks**.
- Calculates the **percentage** of marks.
- Determines the **grade** based on the percentage:
  - Grade A: 90–100%
  - Grade B: 80–89%
  - Grade C: 70–79%
  - Grade D: 60–69%
  - Grade F: Below 60%
